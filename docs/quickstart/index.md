# Quickstart

speek provides three commands, each suited to a different workflow.

::::{grid} 1 1 3 3
:gutter: 2

:::{grid-item-card} speek
:link: speek
:link-type: doc
Classic Rich table. Full partition breakdown, per-user rankings, upcoming release windows.
:::

:::{grid-item-card} speek-min
:link: speek-min
:link-type: doc
Lightweight one-shot or live bars. No disk I/O, fast startup, colour-coded GPU utilisation.
:::

:::{grid-item-card} speek-max
:link: speek-max
:link-type: doc
Full interactive TUI. Tabs, themes, live refresh, job management, log viewer, job submission.
:::
::::

```{toctree}
:hidden:

speek
speek-min
speek-max
```
