# Themes

`speek-max` ships with several built-in themes. Press `Ctrl+T` to cycle through them, or set one at launch with `--theme`.

## Available themes

| Name | Style |
|------|-------|
| `textual-dark` | Default dark theme |
| `textual-light` | Light background |
| `dracula` | Purple-tinted dark |
| `monokai` | Green-accented dark |
| `solarized-dark` | Warm dark |
| `solarized-light` | Warm light |
| `gruvbox` | Retro earthy dark |
| `catppuccin-mocha` | Muted pastel dark |
| `catppuccin-latte` | Muted pastel light |
| `nord` | Cool blue-grey dark |
| `amber` | Warm amber-toned dark |

## Setting a theme at launch

```bash
speek-max --theme dracula
```

The theme name is case-insensitive.

## Switching themes at runtime

Press `Ctrl+T` to advance to the next theme in the list. The UI redraws immediately — all colour tokens (bars, state indicators, user ranks) update automatically.

## Config tab

The **Config** tab (key `5`) provides a visual theme picker with a preview of each option.
